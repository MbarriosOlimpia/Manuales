# Manuales

Repositorio centralizado de OlimpIA que contiene documentación técnica, manuales de uso, tutoriales y especificaciones de APIs. Su propósito es facilitar el acceso a los recursos de referencia para los equipos de desarrollo y operaciones.

---

## 📁 Estructura del repositorio

```
Manuales/
├── APIs/          → Documentación de APIs (App Móvil y MiLicencia)
├── Manuales/      → Manuales técnicos internos
└── Tutoriales/    → Guías e instructivos de herramientas
```
